# Change Filament
## Preparation
1. Turn on 3D printer and wait until the green background is on.
2. Select `Menu` > `Extruder`.
4. Set Extruder Temperature:
   |Filament|Temperature|
   |---|---|
   |PLA|210 °C|
   |TPU|230 °C|
5. Wait for the desired temperature.
